　　原型开发好以后，通常需要一些展示数据。如果没有实际数据，怎样为 Java 程序生成一些有意义的随机数据？

　　我用过 UUID 和 ThreadLocalRandom 来生成随机数，但是能力有限，无法提供特定领域的随机数据。于是，我开始寻找更好的解决方案。最后找到了 Faker 这个好东西，幸运的是有 Java 版本：Java Faker。不仅使用方便，而且可以设置多个领域，几秒钟内生成需要的随机数。

  • 地址
  • 商务
  • …

  • 尤达
  • 查克·诺里斯
  • 哈利·波特
  • 指环王
  • …
